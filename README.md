# Xiaomi Mimo 2.5 Pro 模型前端能力展示

这是一个围绕 `xiaomi mimo2.5pro` 构造的前端能力展示网站，重点用于呈现该模型在网页生成、视觉设计、交互组织和动态效果实现方面的能力。

站点重点展示：

- WebGL / Three.js 动效与空间视觉
- CSS 高级动画与滚动驱动效果
- SVG 路径变形与图形过渡
- Canvas 生成式图形
- Web Audio 实时可视化
- 纯前端组合式交互编排能力

## 在线访问

演示地址：

<https://mimo.tiya.qzz.io>

## 模型能力

这个项目用于集中展示 `xiaomi mimo2.5pro` 在前端方向的能力，包括：

- 复杂页面结构生成能力
- 多模块视觉统一能力
- 动效与交互设计能力
- 原生前端技术组合能力
- 展示型页面创意表达能力

## 本地开发

安装依赖：

```bash
npm install
```

启动开发环境：

```bash
npm run dev
```

## 生产构建

```bash
npm run build
```

本项目构建产物默认输出到 `dist/`。

## 本地预览

```bash
npm run preview
```
